resources/Config.properties notes:
MODEL and VIEW fields are the fields that we are using to change between a
JavaFX GUI or an console output and Database or File storage

MODEL
1 -> MySQL Database
0 -> Config.properties

VIEW
1 -> JavaFX GUI
0 -> Console output

WARNING!!!! In case of bad parameter notation the default optios are File and 
Console
